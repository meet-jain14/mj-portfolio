# MJ Portfolio — Master Context

## Project Vision

This is NOT a traditional developer portfolio.

The goal is to create:
- an immersive interactive digital experience
- a futuristic AI-engineer identity
- a cinematic web experience
- a premium product-like portfolio

The portfolio should psychologically communicate:
- “This person can build real products.”
- “This person understands UX deeply.”
- “This person is an AI engineer.”

The website should feel:
- cinematic
- immersive
- atmospheric
- futuristic
- elegant
- interactive
- intimidatingly polished

NOT:
- generic
- template-based
- corporate
- childish
- cluttered
- over-neon cyberpunk

- Never ask AI to build entire portfolio at once
- Build isolated systems one by one
- Prioritize architecture before visuals
- Prioritize polish over quantity
- Maintain performance throughout development
